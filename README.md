# MathTwin

MathTwin is a symbolic mathematical analysis tool built with:

- SymPy
- PyQt6
- PyQtGraph
- NumPy
- Matplotlib

## Features

- Function plotting
- Derivatives
- Integrals
- Limits
- Domain analysis
- Parity detection
- Asymptote detection
- Real-time graphing

## Installation

```bash
pip install -r requirements.txt
python main.py
```
